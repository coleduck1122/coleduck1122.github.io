第2章——数据类型

  1. (单选题) 利用print()格式化输出,哪个选项用于控制浮点数的小数点后两位输出?
    A {.2}
    B {:.2}
    C {:.2f}
    D {.2f}

    答案：C
    解析：在Python中，可以使用{}占位符和format()方法来实现格式化输出。
    A与D没有冒号，未指定参数位置，语法错误。而C满足题目要求
    B的输出有些奇怪，小数位数足够的时候，是保留数字的有效位数，可以试下下面两个语句
    print(“{:.5}”.format(314.15926)) print(“{:.3}”.format(314.15926))
    详解：https://www.runoob.com/python/att-string-format.html

  1. (单选题) 对于字符串 s = ‘jwtent’,下列表达式的输出和其他三个不同的是( )
    A s[::-5]
    B s[0] + s[-1]
    C s[::-1][::-5]
    D s[::5]

    答案：A
    解析：在Python中，可以使用s[start:end:step]来实现切片操作，其中，start表示起始位置（默认为0），end表示结束位置（默认为字符串长度），step表示步长（默认为1）。
    A. s[::-5] 表示从右向左每隔五个字符取一个字符，结果是’tj’
    B. s[0] + s[-1] 表示取第一个字符和最后一个字符并拼接起来，结果是’jt’
    C. s[::-1][::-5] 表示先反转字符串’tnetwj’再从右向左每隔五个字符取一个字符，并拼接起来，结果是’jt’
    D. s[::5] 表示从左向右每隔五个字符取一个字符，并拼接起来，结果是’jt’
    详解：https://zhuanlan.zhihu.com/p/79541418

  1. (单选题) 下列运算符使用错误的是( )
    A 3*’abc’
    B [1,2,3]+[4,5,6]
    C (-10) % (-3)
    D 1+’a’

    答案：D
    解析：python是强类型语言，无法实现int和string的相加，故D错误
    其他项的结果分别为 abcabcabc , [1,2,3,4,5,6] , -1

  1. (单选题) 关于<模板字符串>.format(<逗号分隔的参数>)中的内容描述,正确的选项是:
    A 模板字符串里的格式控制信息语法格式是:{<参数序号>:<格式控制标记>},参数序号是从1开始
    B 格式控制信息{:*^10}表示这个位置是最大长度为10的整数
    C 格式控制信息{2:0>7f}里面的‘f’表示这个位置是一个整数
    D 逗号分隔的参数可以是变量,也可以是函数

    答案：C
    解析：A错误，参数序号从0开始。
    B，{:*^10}表示控制第一个参数以宽度为10的域居中输出其余空位补星号，且长度大于等于10的内容原样输出
    print(“{:*^10}”.format(314.15)) -> **314.15**
    print(“{:*^10}”.format(314.15111111)) -> 314.15111111

  1. (单选题) 关于 a and b 的描述错误的是 ( )
    A 若 a=False, b=False 则 a and b 的值为 False
    B 若 a=False, b=True 则 a and b 的值为 False
    C 若 a=True, b=False 则 a and b 的值为 True
    D 若 a=True,b=True 则 a and b 的值为 True

    答案：C

  1. (单选题) 关于Python数据类型的描述,正确的选项是:
    A 运算符+、-、*、/ 等跟赋值符号 = 相连,形成增强赋值操作符
    B 内置函数divmod(x,y)的运算结果是两个整数:x除y的整数商以及余数
    C 函数ord(x)是返回字符串x对应的Unicode编码
    D 函数lower(x)是将字符串x的首字母小写

    答案：A

  1. (单选题) 关于Python语言的注释语句的描述,正确的是:
    A ’’’开头的语句也表示注释,用法跟#一样
    B #之后的语句被解释器解释,但不执行
    C 以#开头的语句是注释
    D #注释符可以注释多行

    答案：C

    B 整数
    C 字符串
    D 实数

    答案：D

  1. (单选题) 以下关于程序设计语言的描述,正确的选项是:

  1. (单选题) 一般情况下, 整数用十进制表示,如果用其他进制表示一个数,错误的描述选项是:()

    D s4=frozenset((3,2,1))

    D 其他选项都不正确

    A 缩进是非强制的,仅为了提高代码可读性`
    B 缩进统一为4个空格
    C 缩进在程序中长度统一且强制使用
    D 缩进可以用在任何语句之后,表示语句间的包含关系

  1. (单选题) 对于表达式 a = [3, 4, 5, 6, 7, 9, 11],以下输出结果正确的是( )

    B print(a[0:0]) => [3]
    C print(a[1:-1]) => [4,5,6,7,9,11]

    答案：A
