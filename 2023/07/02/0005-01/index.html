搭建React Native环境（IntelliJ IDEA为例）

发表于2023-07-02|更新于2023-07-02|React Native
|阅读量:

可参考官方文档
https://www.reactnative.cn/docs/environment-setup

第 1 步 - 安装Node

注意 Node 的版本应大于等于 16，安装完 Node 后建议设置 npm 镜像（淘宝源）以加速后面的过程（或使用科学上网工具）。
注意：强烈建议始终选择 Node 当前的 LTS （长期维护）版本，一般是偶数版本，不要选择偏实验性质的奇数版本。
注意：不要使用 cnpm！cnpm 安装的模块路径比较奇怪，packager 不能正常识别！

参考文档
https://blog.csdn.net/wpp555/article/details/107839016

官网 https://nodejs.org/en/download/
中文官网 http://nodejs.cn/download/

以默认路径安装在C:\Program Files\nodejs下当然最省事
如果要修改默认路径
npm config set prefix <你自定义的nodejs安装路径>\node_global
npm config set cache <你自定义的nodejs安装路径>\node_cache
然后需要配置一下系统环境变量，具体细节可去参考别的文档，有很多教程
可用 npm config get prefix 或者 npm root -g 来查看npm的全局安装目录
来确认自己是否修改成功

参考文档
https://blog.csdn.net/Marvin_996_ICU/article/details/106240065

React Native 需要 Java Development Kit [JDK] 11。
低于 0.67 版本的 React Native 需要 JDK 1.8 版本（官方也称 8 版本）。
